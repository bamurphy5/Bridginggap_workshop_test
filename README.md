Welcome! This repo is for the 'Bridging the Gap: Flux Data Meets Land Surface Models' workshop organized by members of the FLUXNET Early Career Network, taking place August 6-9 2025 in Berkeley, CA.

The model-based workshop tutorials and information about setting up Docker are in the workshop wiki page: https://github.com/bamurphy5/Bridginggap_workshop_test/wiki

All files required to run ELM simulations and complete the model-based workshop tutorials are shared via a Docker container, refer to the wiki for more information.
